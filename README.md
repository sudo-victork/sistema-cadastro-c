# Sistema de Cadastro em C

Projeto simples desenvolvido em linguagem C para cadastro de clientes via terminal.

## Funcionalidades
- Cadastro de clientes
- Listagem de clientes
- Busca por ID
- Ordenação por ID
- Remoção de clientes

## Tecnologias
- Linguagem C
- Git

## Como executar
1. Compile o código:
   ```
   gcc main.c -o programa
   ```
2. Execute:
   ```
   ./programa
   ```

## Objetivo
Praticar conceitos da disciplina de Algoritmos, como uso de structs, vetores e funções em linguagem C.
 desenvolvido para fins acadêmicos e prática pessoal.
